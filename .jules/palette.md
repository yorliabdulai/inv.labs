## 2024-05-19 - Accessible Close Buttons
**Learning:** Found several floating and dynamic components (like `MilestoneCelebration`, `FoundingMemberBanner`, `TradeModal`, `OnboardingTour`) that rely on Lucide React icons (e.g., `X`, `ChevronLeft`) wrapped in `button` tags but lacking `aria-label` attributes and proper keyboard focus states, effectively making them inaccessible to screen readers and keyboard users.
**Action:** Always verify that interactive icon-only control elements (especially within modals, banners, and tours) include descriptive `aria-label`s and `focus-visible:outline-none focus-visible:ring-2 focus-visible:ring-ring focus-visible:ring-offset-2` (or similar appropriate ring color) to ensure they are fully navigable and understandable without visual context.
